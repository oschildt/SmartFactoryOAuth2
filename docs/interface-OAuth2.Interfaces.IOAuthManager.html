Method: authenticateUser

public authenticateUser(array $credentials, array &$response) : boolean
Authenticates the user based on the specified credentials and writes the response.

Details

It should use \OAuth2\Interfaces\IUserAuthenticator for the authentication.

Parameters

$credentials

Type: array

The credentials for the authentication. They may vary depending on implementation.

$response

Type: array

The response upon successful authentication. The expected properties are:

  • $response["user_id"] - id of the user.

  • $response["jwt_access_token"] - jwt access token generated upon successful authentication. JWT access token is the access token encoded and signed by the JWT standard approach.

  • $response["refresh_token"] - refresh token generated upon successful authentication.

  • $response["access_token_ttl"] - time to live in seconds for the access token.

  • $response["refresh_token_ttl"] - time to live in seconds for the refresh token.

Returns

Type: boolean

The method should return true upon success, otherwise false.

Throws

\Exception

It might throw an exception in the case of any system errors.


\OAuth2\InvalidCredentialsException

It should throw the InvalidCredentialsException if the authentication fails.


\OAuth2\MissingParametersException

It should throw the MissingParametersException if any required paramters are empty.

Method: invalidateClient

public invalidateClient(string $user_id, string $client_id, string $refresh_token ) : boolean
Invalidates all token records for the client.

Details

The invalidation by the client id should be asserted by a valid user id and a valid refresh token. If the refresh token is expired, the invalidation can be done only after authentication.

The invalidation is done by using \OAuth2\Interfaces\ITokenStorage::deleteTokenRecordByKey() with the key 'client_id'.

Parameters

$user_id

Type: string

The user id for which the refresh token was issued.

$client_id

Type: string

The client id for which the refresh token was issued.

$refresh_token

Type: string

The refresh token generated upon successful authentication.

Returns

Type: boolean

The method should return true on successful invalidation, otherwise false.

Throws

\Exception

It might throw an exception in the case of any errors.


\OAuth2\InvalidTokenException

It should throw the InvalidTokenException if the refresh token is invalid.


\OAuth2\MissingParametersException

It should throw the MissingParametersException if some required paramters are empty.

Method: invalidateJwtAccessToken

public invalidateJwtAccessToken(string $jwt_access_token) : boolean
Invalidates the token record for the jwt access token.

Details

The invalidation is done by using \OAuth2\Interfaces\ITokenStorage::deleteTokenRecordByKey() with the key 'access_token'.

The origin access token is extracted from the jwt payload.

Parameters

Type: string

The jwt access token generated upon successful authentication.

Type: boolean

The method should return true on successful invalidation, otherwise false.

Throws

Refreshes the access and refresh tokens by using the valid refresh token.

$user_id

Type: string
